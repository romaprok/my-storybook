# Typography Component

A flexible typography component that provides consistent text styling across your application.

## Features

- Multiple text variants (h1-h6, body1, body2, caption, overline)
- Text decoration options (underline, strikethrough, overline)
- Text transformation (uppercase, lowercase, capitalize)
- Line height control
- Letter spacing control
- Text truncation (single and multi-line)
- Text shadow effects
- Font family options
- Custom font support
- Color variants
- Text alignment
- Font weight options

## Usage

```tsx
import { Typography } from './Typography';

// Basic usage
<Typography variant="h1">Main Heading</Typography>
<Typography variant="body1">Regular paragraph text</Typography>

// With styling
<Typography 
  variant="h4" 
  color="primary" 
  align="center" 
  weight="bold"
>
  Styled Heading
</Typography>

// With text effects
<Typography 
  variant="body1" 
  decoration="underline"
  transform="uppercase"
  letterSpacing="wide"
>
  Styled Text
</Typography>
```

## Accessibility

The Typography component follows accessibility best practices:

- Uses semantic HTML elements
- Maintains proper heading hierarchy
- Includes appropriate ARIA attributes
- Ensures sufficient color contrast
- Supports screen readers

## Best Practices

1. Use appropriate heading levels (h1-h6)
2. Maintain consistent text hierarchy
3. Choose readable font sizes and line heights
4. Ensure sufficient color contrast
5. Use truncation carefully to avoid losing important content

## CSS Customization

The Typography component can be customized through:

1. Built-in props for common styling needs
2. CSS Module classes for component-specific styles
3. Custom font families for unique typography
